## Descripción

A través del ejercicio se pudieron crear los diferentes niveles para el support handle, que permitieron entender el modelo de programación titulado "Chain of Responsability" designando diferentes niveles para las responsabilidades. Cada una de las clases de niveles heredan de Support Handle, para reescribir su método y determinar si debe pasar o no al siguiente nivel.
Adicionalmente se integró otro nivel de responsability, denominado Urgent, que permite ver que al extender no existen modificaciones a las otras clases.
